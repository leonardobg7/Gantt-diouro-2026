# ObraFlow Planner

Base inicial de um SaaS de planejamento de obras com gráfico de Gantt interativo, importação de planilhas e calendário brasileiro.

## Stack
- React
- TypeScript
- Vite
- Zustand
- XLSX

## Rodar localmente
1. npm install
2. npm run dev

## Objetivo
- tabela de tarefas com hierarquia
- gráfico de Gantt sincronizado
- cálculo em dias úteis
- caminho crítico
- importação de XLSX e CSV
- persistência local

## Estrutura principal
- src/app
- src/components
- src/modules/planner
- docs
- prompts
- public/holidays
