# Gantt Rules

## Layout
- grade à esquerda
- Gantt à direita
- rolagem vertical sincronizada

## Regras
- summaries agregam filhos
- tasks podem ser movidas e redimensionadas
- milestones têm visual próprio
- fins de semana destacados
- linha de hoje visível
- caminho crítico destacado

## Vínculos
- suportar FS, SS, FF, SF
- suportar lag
- rejeitar ciclos

## Visual
- tema dark premium
- azul dominante
- barras arredondadas
- setas discretas
- timeline legível
