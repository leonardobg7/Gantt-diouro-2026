# Product Overview

ObraFlow Planner é um SaaS de planejamento de obras inspirado em MS Project, com foco em:
- tabela editável
- gráfico de Gantt premium
- importação de planilhas
- calendário brasileiro
- caminho crítico

## MVP
- React + TypeScript + Vite
- persistência local
- projeto mock
- hierarquia de tarefas
- gráfico de Gantt sincronizado
- predecessoras FS, SS, FF, SF
- fins de semana e feriados
- importação de XLSX e CSV

## Fora do escopo inicial
- autenticação
- banco remoto
- multiusuário
- colaboração em tempo real

## Requisitos visuais
- tema dark premium
- azul com apoio de roxo
- barras elegantes
- setas discretas
- boa legibilidade

## Requisitos técnicos
- lógica de negócio fora da UI
- estrutura legível por IA
- arquitetura pronta para backend futuro
